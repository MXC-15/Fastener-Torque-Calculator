# Fastener-Torque-Calculator
This Python script is a specialized engineering tool designed to calculate the appropriate installation torque for a threaded fastener.Key Engineering Functions:
User Input: The script prompts the user for the three critical physical parameters: bolt diameter (mm), material proof stress (MPa), and the nut factor/friction coefficient K.
Unit Conversion: It automatically handles the conversion from millimeters to meters to ensure calculations align with standard SI unit force measurements.
Preload Estimation: It calculates the target clamping force (preload) using the engineering design standard of 75% of the bolt's proof strength multiplied by the cross-sectional area.
Torque Calculation: It applies the standard industry formula T = K * D * F to determine the final installation torque required to achieve the desired preload.
This tool is an excellent example of how simple procedural programming can automate repetitive mechanical engineering tasks and improve design consistency.
